# 탭 5 — 오류
> 오류 발생 시 자동 기록. 탭바 아이콘에 빨간 숫자 배지 표시. 팝업 없음.

## 테이블 컬럼
| 열 | 내용 |
|----|------|
| 발생시각 | YYYY.MM.DD HH:MM |
| 위치 | 탭명 + 기능명 (예: 탭1 > AI 파싱) |
| 오류 내용 | 에러 메시지 요약 |
| 해결 상태 | 미해결(빨강) / 해결(초록) |

## 오류 유형별 색상
- AI 파싱 오류: #FF9500 주황
- API 수집 실패: #FF9500 주황
- Sheets 쓰기 실패: #FF3B30 빨강
- 인증 실패: #6E6E73 회색

## 상태별 화면
오류 있음: 테이블 표시
오류 없음: "현재 기록된 오류가 없습니다" + 초록 아이콘
