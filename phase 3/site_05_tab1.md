# 탭 1 — 거래 입력
> 4단계 플로우. 상단 진행바(3px) 상태 표시.

## Step 1 — 입력
자연어 입력창 (placeholder: "오늘 삼성전자 10주 8만원에 매수했어")
입력 후 버튼 #0071E3 활성화

## Step 2 — AI 해석 중 (Gemini API)
버튼 비활성 + 스피너 · 진행바 2/4

## Step 3 — 확인 카드
| 항목 | 수정 가능 |
|------|---------|
| 날짜·종목·티커 | ✓ |
| 수량·단가·총금액 | ✓ |
| 투자일지 메모 | ✓ 자유 입력 (길이 제한 없음) |
| 거시경제 스냅샷 | ✗ 참고용 |

버튼 "확인 후 저장" → #34C759

## Step 4 — 완료
성공 메시지 + 포트폴리오 탭 바로가기

## 오류 처리
AI 파싱 실패 → Step 1 복귀 + 재입력 안내
저장 실패 → 탭 5 자동 기록 + 오류 배너
