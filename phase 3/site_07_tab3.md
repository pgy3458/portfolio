# 탭 3 — 거래 이력
> 소스: RAW_DATA. 최신 거래 상단. 탭 진입 시 자동 갱신.

## 필터 바 (상단 고정)
종목명 검색 · 자산군 드롭다운 · 기간 선택 · 매수·매도 토글

## 테이블 컬럼
| 열 | 표시 | 비고 |
|----|------|------|
| 날짜 | YYYY.MM.DD | 최신순 |
| 종목 | 종목명 + 티커 | |
| 자산군 | 배지 | 색상 구분 |
| 수량 | 숫자 | |
| 단가 | 원화 | |
| 총금액 | 원화 | |
| 구분 | 매수/매도 배지 | |

## 행 클릭 시

→ 거시경제 스냅샷 + 투자일지 메모 확인
