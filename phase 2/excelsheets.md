# excelsheets — Google Sheets 구조 정의

## Sheet 1: RAW_DATA (거래이력)

> 역할: 모든 거래 원본 기록. 나머지 시트의 데이터 소스.
> 오류 행은 Z열 기준 빨간 색상 표시.

| 열 | 컬럼명 | 타입 | 자동계산 | 설명 |
|----|--------|------|---------|------|
| A | 날짜 | DATE | 수동 | 거래 발생일 |
| B | 종목명 | TEXT | 수동 | 삼성전자, Apple 등 |
| C | 티커 | TEXT | 수동 | 005930, AAPL 등 |
| D | 국가/시장 | TEXT | 수동 | KR / US / JP / CN |
| E | 자산유형 | TEXT | 수동 | 주식 / ETF / 현금 |
| F | 자산군 | TEXT | 수동 | 성장주 / 가치주 / 채권 / 원자재 |
| G | 기관명 | TEXT | 수동 | 키움 / 미래에셋 / IB 등 |
| H | 액션 | TEXT | 수동 | 매수 / 매도 |
| I | 수량 | NUMBER | 수동 | 거래 수량 |
| J | 단가 | NUMBER | 수동 | 거래 단가 (원화 기준) |
| K | 총금액 | NUMBER | 자동 (I×J) | 수량 × 단가 |
| L | 통화 | TEXT | 수동 | KRW / USD / JPY |
| M | 투자일지/메모 | TEXT | 수동 | 거래 이유, 자유 기록 |
| N | 원문입력 | TEXT | 자동 | AI에 입력한 자연어 원문 |
| O | USD/KRW | NUMBER | 자동수집 | 거래 시점 환율 |
| P | USD/JPY | NUMBER | 자동수집 | 거래 시점 환율 |
| Q | USD/CNY | NUMBER | 자동수집 | 거래 시점 환율 |
| R | 한국금리 | NUMBER | 자동수집 | 거래 시점 기준금리 |
| S | 미국금리 | NUMBER | 자동수집 | 거래 시점 기준금리 |
| T | 일본금리 | NUMBER | 자동수집 | 거래 시점 기준금리 |
| U | KOSPI | NUMBER | 자동수집 | 거래 시점 지수값 |
| V | S&P500 | NUMBER | 자동수집 | 거래 시점 지수값 |
| W | NASDAQ | NUMBER | 자동수집 | 거래 시점 지수값 |
| X | Nikkei | NUMBER | 자동수집 | 거래 시점 지수값 |
| Y | 상해종합 | NUMBER | 자동수집 | 거래 시점 지수값 |
| Z | 오류여부 | TEXT | 자동 | 정상 / 오류 (오류 행 빨간 표시) |

## Sheet 2: PORTFOLIO (포트폴리오 현황)

> 역할: RAW_DATA 기반 자동 계산된 현재 보유 현황.
> 섹션 순서 = 스프레드시트 위→아래 배치 순서.

### [섹션 0] 차트 영역 ← 최상단 고정

| 차트명 | 종류 | 데이터 소스 | 설명 |
|--------|------|------------|------|
| 투자 국가별 보유 현황 | 원형(Pie) | 섹션D D열 기반 | 국가별 평가금액 합산 비중 |
| 국가별 투자 종목 현황 | 원형(Pie) | 섹션D D열 기반 | 국가별 보유 종목 수 비중 |

> 두 차트 나란히 배치 (좌: 금액 기준 / 우: 종목 수 기준)

---

|------|------|---------|

| USD/KRW / USD/JPY / USD/CNY | NUMBER | 자동수집 |
| 현재가 · 등락률 · 52주 고가 (지수별) | NUMBER | 자동수집 |

| 보유종목 수 / 자산군 수 | NUMBER | 자동 |
